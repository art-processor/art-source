1. You can run the examples with the batch file accompanied with every example.
2. Please install XVCL processor to C:\xvcl. Or else please change the batch file 
   included for every exmaple, and also the DTD directory in every x-frame accordingly.
   You can use some tools to automate the changes of directory.
3. As some examples will use C:\xvclEx\, it is safe that you put all
   the examples in C:\xvclEx\. Or else please change every x-frame which uses this
   directroy accordingly.
